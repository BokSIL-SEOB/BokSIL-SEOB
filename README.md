## 🚀 Main Project

### FanLink (Jan 2026 – Feb 2026)
팬과 아티스트를 연결하는 플랫폼 프로젝트입니다.  
라이브 스트리밍, 다시보기, 미디어 업로드/처리, 팬 참여 기능 등을 다루는 서비스를 팀 프로젝트로 개발했습니다.

#### 담당 영역
- Replay(다시보기) 도메인 백엔드 구현
- `media_asset` 도메인 설계 및 구현
- `music_video` 도메인 백엔드 구현
- `ivs` 기반 라이브 재생/녹화 이벤트 연계 처리
- AWS 기반 미디어 저장 및 처리 흐름 설계
- 배포 자동화 및 운영 구조 구성

#### 주요 경험
- S3 Presigned URL 기반 업로드 흐름 설계
- CloudFront를 활용한 미디어 제공 구조 구성
- IVS 라이브 스트리밍 및 녹화 이벤트 연계 구조 학습 및 적용
- MediaConvert 기반 VOD/HLS 처리 흐름 설계
- EventBridge → SQS → Backend Consumer 구조 이해 및 적용
- GitHub Actions → ECR → SSM → EC2 기반 배포 자동화 경험
- Parameter Store를 활용한 서버 환경변수 관리 경험

#### 프로젝트를 통해 배운 점
이 프로젝트를 통해 단순한 기능 구현을 넘어,  
서비스가 실제 환경에서 **어떻게 저장되고, 처리되고, 배포되고, 운영되는지**를 함께 고민하게 되었습니다.

백엔드 개발과 인프라의 경계를 함께 이해하는 경험을 하면서,  
문제 해결 범위를 코드 내부에만 두지 않고 서비스 운영 관점까지 확장하게 되었습니다.
